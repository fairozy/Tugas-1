    Fuzzy C-Means¶

    Pengertian¶

    ​ Fuzzy C-Means (FCM) adalah suatu teknik pengelompokan data yang keberadaan tiap-tiap data dalam suatu kelompok ditentukan oleh nilai atau derajat keanggotaan tertentu dan teknik ini pertama kali diperkenalkan oleh Jim Bezdek pada tahun 1981. Fuzzy C-Means menerapkan pengelompokan fuzzy, dimana setiap data dapat menjadi anggota dari beberapa cluster dengan derajat keanggotaan yang berbeda-beda pada setiap cluster. Fuzzy C-Means merupakan algoritma iteratif, yang menerapkan iterasi pada proses clustering data. Tujuan dari Fuzzy C-Means adalah untuk mendapatkan pusat cluster yang nantinya akan digunakan untuk mengetahui data yang masuk ke dalam sebuah cluster.

    Algoritma FCM secara lengkap diberikan sebagai berikut:

    1. Tentukan :

    a. Matriks X beukuran n x m, n = jumlah data yang akan di cluster, m = jumlah variable (kriteria)

    b. Jumlah cluster yang akan dibentuk (C >= 2)

    c. Pangkat (bobot w > 1)

    d. Maksimum iterasi

    e. Kriteria penghentian

    1. Bentuk matriks partisi awal U (derajat keanggotaan dalam cluster), matriks partisi awal biasanya dibuat secara acak.

    1. Hitung pusat cluster V untuksetiap cluster

    1. Perbaiki derajat keanggotaan setiap data pada setiap cluster

    1. Tentukan kriteria penghentian iterasi, yaitu dengan cara perubahan pada matriks partisi pada iterasi sekarang dan iterasi sebelumnya.

#Membangkitkan nilai random dari nilai asli menggunakan rumus rata-rata dan standart deviasi:
# bilangan random untuk cluster
norm = [] # miu
def bil_random():
    for i in value:
        n_v = s.mean(i)/s.stdev(i)
        temp=[]
        for j in i:
            temp.append(j-n_v)
        norm.append(temp)
bil_random()

#Selanjutnya fungsi clus, di fungsi ini akan menghitung fungsi objektif, update derajat keanggotaan, bagian code yang diberi comment (#).
nilai_pusat_cluster1=[]
nilai_pusat_cluster2=[]
jumlah_setiap_cluster=[]
v=[2.736,3.306,7.860,6.596]
c=[] #untuk nilai c

def clus():
    count_collumn = 0
    count_row = 0
    for i in norm:
        count_collumn+=1
        temp=[]
        for j in i:
            count_row +=1
            j = j**2
            temp.append(j)
        nilai_pusat_cluster1.append(temp)

    #menghitung pusat cluster

    for x in range(count_collumn):
        temp1=[]
        for y in range(int(count_row/count_collumn)):
            nilai_sementara=nilai_pusat_cluster1[x][y]*value[x][y]
            temp1.append(nilai_sementara)
        nilai_pusat_cluster2.append(temp1)
        jumlah = sum(temp1)
        jumlah_setiap_cluster.append(jumlah)

    nilai_akses=0
    #menghitung fungsi objektif

    for q in range(count_collumn):
        temp2=[]
        if(nilai_akses == 3):
            nilai_akses = 0
        for w in range(int(count_row/count_collumn)):
            value_c=((value[q][w]-v[nilai_akses])**2)*norm[q][w]
            temp2.append(value_c)
        nilai_akses += 1 
        c.append(temp2)

    #menghitung c+cn
    csum=[]
    for cc in  range(count_collumn):
        next = cc + 1
        temp3=[]
        for cr in range(int(count_row/count_collumn)):
            if (next<count_collumn):
                value_c= c[cc][cr]+c[next][cr]
                temp3.append(value_c)
            else:
                next=cr
        csum.append(temp3)


    #jumlah nilai c
    jumlah_fungsi_objektif=sum(temp3)

    #update derajat keanggotaan
    pangkat=-1/w-1
    for dc in range(count_collumn):
        if(nilai_akses >= 3):
            nilai_akses = 0
        for dr in range(int(count_row/count_collumn)):
            norm[dc][dr] = ((value[dc][dr]-v[nilai_akses])**2)**pangkat/((value[dc][dr]-v[cluster])**2)**pangkat
        nilai_akses += 1

    #Hasil
    iterasi_kolom1=1
    for iterasi1 in norm:
        for ke1 in iterasi1:
            print("Kolom ke : ",iterasi_kolom1,"Baris ke : ",ke1)
        iterasi_kolom1 += 1
    print("========================================")
    # untuk hasil kluster 
    iterasi_kolom=1
    for iterasi in csum:
        for ke in iterasi:
            print("Kluster ke : ",iterasi_kolom,"Data ke : ",ke)
            if (ke<1):
                nilai_nol.append(ke)
            elif(ke>1):
                nilai_satu.append(ke)

        iterasi_kolom += 1
    print("banyak derajat keanggotaan yang bernilai 1 ",len(nilai_satu),"banyak derajat keanggotaan yang bernilai 0",len(nilai_nol))
